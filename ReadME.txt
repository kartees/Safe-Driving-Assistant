Safe Driving Assistant

A high level system application with python GUI TKinter

-- REQUIREMENTS TO RUN THE CODE --

install python3
install tensorflow2.0
install keras
install openCV2
install pygame
install matplotlib, numpy, pandas
and some utilities

Run the main.py using "python main.py" in the command prompt or terminal

For authentication, use username as "driver" and password "root"

Then the home page appears and can select any detections simultaneously also

A new window will be opened for each detections

Hardware Requirements

GPU 2gb for better perfomance or no need compulsorily
Front Camera on PC 
Good processor ( better than or equal to i3 processor /Ryzen 3)
RAM 8gb (4gb minimum)
